# HR Dashboard Creation – Power BI, Excel, SQL

## Project Overview
The HR Dashboard Creation project is designed to analyze and visualize key workforce metrics using Power BI, Excel, and SQL. The dashboard helps HR teams and management make data-driven decisions by providing clear insights into employee performance, attrition, and recruitment trends.

This project simulates a real-world HR analytics system used in mid to large-scale organizations.

## Tools & Technologies Used
- Power BI – For interactive data visualization and dashboards
- Microsoft Excel – For data storage, cleaning, and preprocessing
- SQL – For database creation, querying, and workforce analysis

## Key Features
- Interactive HR dashboard with dynamic filters
- Employee performance analysis
- Attrition rate monitoring
- Recruitment and hiring trend analysis
- Department-wise workforce distribution
- SQL-based data analysis for reporting

## Dataset Description
The dataset includes employee details such as:
- Employee ID and Name
- Department and Role
- Date of Joining
- Performance Rating
- Employment Status (Active, Resigned, On Notice)

Data is maintained in Excel and imported into SQL and Power BI for analysis.

## Project Structure
- PowerBI: Contains the Power BI dashboard file
- Excel: Contains the HR dataset
- SQL: Includes table creation and analysis queries
- Screenshots: Dashboard preview images

## How to Use
1. Open the Excel file to review the dataset
2. Import the data into a SQL database using the provided scripts
3. Load the data into Power BI
4. Explore the dashboard using filters and visuals

## Use Case
This dashboard can be used by:
- HR Teams
- Managers
- Business Analysts
- Students learning HR Analytics
